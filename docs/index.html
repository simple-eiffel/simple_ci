Overview

simple_ci is a homebrew Continuous Integration tool for Eiffel projects. Builds, tests, and reports on Eiffel project compilations with zero external dependencies.

Part of the simple_* ecosystem of focused, single-purpose Eiffel libraries.

Quick Start

Installation

Add to your ECF file:

<library name="simple_ci" location="$SIMPLE_CI/simple_ci.ecf"/>

Configuration

-- Configure via JSON file
{
  "projects": [
    {"name": "my_app", "ecf": "my_app.ecf", "target": "my_app"},
    {"name": "my_tests", "ecf": "my_app.ecf", "target": "my_app_tests"}
  ]
}

Features

  • Compiles multiple Eiffel projects
  • Runs test targets
  • JSON configuration
  • Build status reporting
  • Zero external dependencies
  • Full Design by Contract support
